# amazon-clone
A front-end clone of Amazon's homepage built using only HTML and CSS. This project focuses on responsive design, layout structuring, and styling techniques that replicate the look and feel of the original e-commerce platform.

Features
Fully responsive homepage layout

Header with logo, search bar, and navigation links

Product listing sections with hover effects

Footer with multiple column links and copyright

Clean and reusable CSS classes for scalability

Technologies Used
HTML5 for structure

CSS3 for design and layout

Folder Structure
text

│
├── index.html
├── style.css
└── README.md
